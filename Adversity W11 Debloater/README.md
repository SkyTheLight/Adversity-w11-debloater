# Adversity - Windows 11 Debloater

A modern, minimalist Windows 11 debloater with black/purple theme. Inspired by Chris Titus Tech's winutil.

## Features

- **Modern GUI** - Black/purple Catppuccin theme
- **App Installer** - Winget + Chocolatey support
- **Tweaks** - Privacy, AI, System, Taskbar, Gaming
- **Gamer Tab** - FPS optimizations, latency reduction
- **Presets** - Standard, Minimal, Gamer, FPS

## Usage

1. Run `Run.bat` as Administrator
2. Or: `powershell -ExecutionPolicy Bypass -File Adversity.ps1`

## Tabs

- **Apps** - Install applications
- **Tweaks** - System optimizations
- **Config** - Configuration
- **Updates** - Windows updates
- **ISO** - Windows ISO tools
- **Gamer** - Gaming optimizations

## Keyboard Shortcuts

- `I` - Install tab
- `T` - Tweaks tab  
- `C` - Config tab
- `U` - Updates tab
- `W` - ISO tab
- `G` - Gamer tab

## Requirements

- Windows 10/11
- PowerShell 5.0+
- Administrator
